# CORE_PROJECT_RULES — Astro methodology core

Этот файл заменяет старое имя `AGENTS.md` внутри `00_CORE`.

Назначение: справочник правил методического ядра для проекта Ракурс ИЖС.

## Как использовать

- Общие правила Codex берутся из глобального `C:\Users\User\.codex\AGENTS.md`.
- Локальный контекст проекта хранится в `PASSPORT_PROJECTS.md`.
- Журнал проекта хранится в `WORKLOG.md` и `PROJECT_JOURNAL.md`.
- Этот файл использовать только как методический справочник, если задача требует правил `00_CORE`.

## Главный принцип

Документ опережает код.

Перед сборкой должны быть актуальны:

- `SITE_ARCHITECTURE.md`;
- `PAGE_HOME.md` / `PAGE_*.md`;
- `DESIGN_SYSTEM.md`.

Если документов нет или они расходятся с кодом, сначала актуализировать документы.
